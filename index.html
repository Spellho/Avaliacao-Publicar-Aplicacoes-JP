1 - Aplicações Web

Aplicações web são as aplicações que são executadas em um navegador web, elas são divididas em estáticas e dinâmicas:

Estáticas - Aplicações estáticas são aquelas que apresetam um conteúdo fixo e que não precisam de um backend para funcionar.

Dinâmicas - Aplicações dinâmicas permitem a interação com o usuário, pois possuem um backend que pode atualizar seu conteúdo em tempo real.

2 - Domínio e DNS

Dominio - O domínio de um site é o seu endereço único, constituido de um nome e uma extensão (.com, .net, .org).

DNS - Domain Name System (Sistema de Nome de Domínios) é um serviço que permite que pessoas acessem sites usando o domínio e não o IP, facilitando a navegação.

3 - Servidores Compratilhados e Dedicados

Compartilhados - Usados por multiplos usuários e empresas ao mesmo tempo, servidores compartilhados são a opção mais barata e simples, pois o custo é dividido entre os usuários e eles te poupam o tempo e esforço de criar seu próprio servidor.

Dedicados - Servidores dedicados são compostos de apenas um computador e têm o propósito de serem utilizados por apenas uma empresa. Dessa forma, todos os custos do servidor são pagos por essa empresa, mas ela também tem controle de todos os recursos e personalizações desse servidor.

4 - Servidores Web

Servidores web são os servidores que contém os arquivos e componentes necessários para o uso de uma aplicação web específica. Eles são divididos em estáticos e dinâmicos:

Estáticos - Utilizados em aplicações estáticas, ou seja, entregam os arquivos diretamente ao usuário.

Dinâmicos - Utilizados em aplicações dinâmicas, servidores dinâmicos processam seus dados de acordo com a requisição antes de envialos.

Como fazer o backup compactado de um projeto com gzip em debian:
"gzip -r -c 'nomedoprojeto' > 'nomedobackup'.gz"

Como restaurar um backup compactado de um projert com gzip em debian:
"gzip -d 'nomedobackup'"

5 - Servidores DB

Um servidor de banco de dados contém e fornece acesso a um ou mais bancos de dados.

Como fazer backup de um banco de dados:
"mysqldump -u'nomedousuario' -p 'nomedobanco' > 'nomedobackup'.sql"

Como restaurar o backup de um banco de dados:
"mysql -u'nomedousuario' -p 'nomedobanco' < 'nomedobackup'.sql"

6 - Servidor SSH

Servidores SSH são servidores que utilizam o protocolo Secure Shell para permitir que usuários acessem e controlem servidore de forma remota

Comando para transferir arquivos para um servidor usando ssh:
"scp 'nomedoarquivo' 'usuario'@'servidor':'caminho'/'destino'/"

7 - Linux

Sistemas Linux são sistemas operacionais que utilizam o núclea Linux. As principais distribuições utilizadas como servidor seb são Ubuntu, Debian e CentOS.

8 - Comandos Linux

ls: lista os arquivos e pastas no diretório atual.
clear: limpa a tela.
cd: entra em um diretório.
pwd: lê o nome do diretório atual.
cat: lê o conteúdo de um arquivo.
mv: move e renomeia arquivos e diretórios.
rm: remove arquivos e diretórios.
touch: cria arquivos vazios.
mkdir: cria diretórios.
df: mostra o espaço ocupado no disco.
top: mostra o uso de memório.
vi: abre o editor vi.
